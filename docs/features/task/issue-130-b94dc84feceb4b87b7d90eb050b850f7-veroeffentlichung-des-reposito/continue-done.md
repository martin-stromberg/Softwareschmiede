Diese Diskrepanz wurde dem Anwender gemeldet; er hat daraufhin die Entfernung des toten Codes gewünscht und die Erstellung eines Tracking-Issues beauftragt: **[Issue #131](https://github.com/martin-stromberg/Softwareschmiede/issues/131) — „Toten Code zu Agentenpaketen entfernen (IAgentPackageService, AgentPackageReader u. a.) inkl. README-Anpassung"**. Die eigentliche Codeentfernung samt Nachziehen der README (Mermaid-Diagramm-Knoten `APL6`/`INL6`, Testverweis) erfolgt als eigener Lifecycle-Durchlauf zu diesem Issue, nicht im Rahmen dieses Dokumentationszyklus.
