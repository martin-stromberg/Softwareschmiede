# BitBucket-Plugin

Das BitBucket-Plugin unterstützt Git-basierte Repositories bei **BitBucket Cloud** (bitbucket.org) und **BitBucket Server/Data Center** (Self-Hosted) als SCM-Provider. Es ermöglicht Repository-Verwaltung, Branch-Operationen, Pull-Request-Erstellung und Jira-Issue-Integration.
