# Programmupdate

Das Programmupdate-Feature ermöglicht dem Anwender, die Softwareschmiede-Anwendung über das grafische Interface zu aktualisieren. Ein Fortschrittsdialog zeigt den Status der Update-Vorbereitung (Download, Entpacken, Update-Vorbereitung) an.
