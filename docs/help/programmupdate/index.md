# Programmupdate

Das Programmupdate-Feature ermöglicht dem Anwender, die Softwareschmiede-Anwendung über das grafische Interface zu aktualisieren. Vor dem Update wird überprüft, ob laufende CLI-Aufgaben das Update blockieren würden; der Anwender wird ggf. gewarnt und kann dann entscheiden, ob er fortfahren möchte. Bei Bestätigung zeigt ein Fortschrittsdialog den Status der Update-Vorbereitung (Download, Entpacken, Update-Vorbereitung) an.
