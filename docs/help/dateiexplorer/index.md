# Dateiexplorer

Der Dateiexplorer ist ein neues Register in der Aufgabendetailansicht, das die lokale Verzeichnisstruktur des geklonten Repositories durchsuchbar macht — mit zwei Modi zur Übersicht über alle Dateien oder nur geänderte Dateien im Branch.
