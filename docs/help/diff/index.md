# Diff-Anzeige

Die Diff-Anzeige zeigt die Unterschiede zwischen zwei Versionen einer Datei, zum Beispiel zwischen dem aktuellen Branch-Stand und dem Basis-Branch. Sie ist in die Aufgabendetailansicht integriert und kann auch als eigenständige Seite aufgerufen werden.
