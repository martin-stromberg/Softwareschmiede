# F028 – Startvalidierung beim Aufgabenstart

## Einleitung

Diese Funktion erklärt, welche Felder vor dem Start ausgefüllt sein müssen.
Sie schafft Klarheit für den täglichen Ablauf.
So starten Sie Aufgaben schneller und vermeiden unnötige Rückfragen.
Der aktuelle Sollzustand ist eindeutig.
Nur das **KI-Plugin** ist Pflicht.

---

## Wer nutzt es?

Diese Übersicht nutzen Fachanwender beim Start einer Aufgabe.
Neue Mitarbeitende nutzen sie als einfache Orientierung.
Stakeholder nutzen sie für klare Arbeitsregeln im Team.

---

## Schritt-für-Schritt-Anleitung

1. Sie öffnen eine Aufgabe im Status **Offen**.
2. Sie prüfen das Feld **KI-Plugin** und wählen einen Eintrag.
3. Sie entscheiden, ob Sie **Agentenpaket** und **Agent** nutzen möchten.
4. Sie lassen optionale Felder leer oder treffen eine Auswahl.
5. Sie klicken auf **KI starten**.

---

## Beispiel

Sie möchten eine kleine Korrektur im Text durchführen.
Sie wählen nur ein **KI-Plugin** aus.
Die Felder **Agentenpaket** und **Agent** lassen Sie leer.
Sie klicken auf **KI starten**.
Die Aufgabe startet ohne weitere Pflichtangaben.

---

## Was passiert im Hintergrund?

Die Oberfläche prüft beim Start zuerst das Pflichtfeld.
Ist kein KI-Plugin gewählt, bleibt der Startknopf gesperrt.
Sind optionale Felder leer, startet die Aufgabe trotzdem.
So bleibt der Ablauf einfach und verlässlich.

---

## Häufige Fragen (FAQ)

**Kann ich ohne Agentenpaket starten?**
Ja. Das Feld ist optional.

**Kann ich ohne Agent starten?**
Ja. Auch dieses Feld ist optional.

**Kann ich ohne KI-Plugin starten?**
Nein. Ohne KI-Plugin ist der Start nicht möglich.

**Gelten diese Regeln auch für Folge-Anweisungen?**
Ja. Das KI-Plugin bleibt die zentrale Pflichtauswahl.
