# F008 – Dashboard

## Einleitung

Das Dashboard ist die Startseite der Softwareschmiede.
Es zeigt Ihnen auf einen Blick alle laufenden und kürzlich abgeschlossenen Aufgaben – aus allen Ihren Projekten zusammen.
So sehen Sie sofort, was die KI gerade erledigt, was erfolgreich abgeschlossen wurde und wo ein Problem aufgetreten ist.
Sie brauchen nicht erst in jedes Projekt einzeln zu navigieren.
Das Dashboard gibt Ihnen den schnellen Einstieg in Ihren Arbeitsalltag.

![Dashboard-Übersicht](../images/F008-dashboard.png)

---

## Wer nutzt es?

Das Dashboard nutzt **jeder**, der die Softwareschmiede öffnet.
Es ist besonders hilfreich, wenn Sie mehrere Projekte gleichzeitig betreuen.
Neue Mitarbeitende erkennen hier auf Anhieb, welche Vorhaben gerade aktiv sind.

---

## Schritt-für-Schritt-Anleitung

### Dashboard öffnen

1. Sie starten die Softwareschmiede in Ihrem Browser.
2. Das Dashboard wird automatisch als erste Seite angezeigt.
3. Sie sehen die Aufgaben aller Ihrer Projekte in einer gemeinsamen Übersicht.

### Status einer Aufgabe ablesen

Die Aufgaben sind farblich markiert. So erkennen Sie den Status auf den ersten Blick:

| Farbe | Bedeutung |
|-------|-----------|
| 🔵 **Blau** | Die KI arbeitet gerade an dieser Aufgabe. |
| 🟢 **Grün** | Die Aufgabe wurde erfolgreich abgeschlossen. |
| 🔴 **Rot** | Bei der Aufgabe ist ein Fehler aufgetreten. |
| ⚪ **Grau** | Die Aufgabe ist offen oder wird manuell vorbereitet. |

### Zu einer Aufgabe navigieren

1. Sie sehen eine Aufgabe im Dashboard, die Sie interessiert.
2. Sie klicken direkt auf den Titel der Aufgabe.
3. Die Detailansicht der Aufgabe öffnet sich sofort.
4. Von dort aus können Sie das Protokoll lesen, die KI starten oder die Aufgabe abschließen.

### Bedeutung der Farben verstehen

- **Blau** bedeutet: Die KI ist aktiv. Warten Sie, bis die Aufgabe abgeschlossen ist.
- **Grün** bedeutet: Alles erledigt. Die Aufgabe kann jetzt freigegeben werden.
- **Rot** bedeutet: Es ist etwas schiefgelaufen. Öffnen Sie die Aufgabe und prüfen Sie das Protokoll.
- **Grau** bedeutet: Die Aufgabe wartet noch auf Ihren Start.

---

## Beispiel

Sie haben drei Projekte aktiv: „Buchungssoftware", „Firmenwebseite" und „Lagerverwaltung".
Sie öffnen die Softwareschmiede und sehen sofort das Dashboard.
Eine Aufgabe in der „Buchungssoftware" wird gerade von der KI bearbeitet (🔵 blau).
Eine Aufgabe in der „Lagerverwaltung" ist fehlgeschlagen (🔴 rot).
Sie klicken direkt auf die fehlgeschlagene Aufgabe und öffnen das Protokoll.
Dort sehen Sie, was schiefgelaufen ist, und können die Aufgabe korrigieren und neu starten.
Für die laufende Aufgabe in der „Buchungssoftware" müssen Sie nichts tun – sie läuft weiter.

---

## Was passiert im Hintergrund?

Das Dashboard fragt beim Öffnen automatisch den aktuellen Stand aller Aufgaben ab.
Sie müssen die Seite nicht manuell aktualisieren.
Sobald eine Aufgabe ihren Status ändert – zum Beispiel von „KI aktiv" zu „Abgeschlossen" – wechselt auch die Farbe automatisch.
Alle Projekte werden gleichzeitig berücksichtigt, ohne dass Sie zwischen ihnen wechseln müssen.

## Häufige Fragen (FAQ)

**Warum sehe ich keine Aufgaben im Dashboard?**
Wenn noch keine Aufgaben angelegt wurden, ist das Dashboard leer. Legen Sie zunächst ein Projekt und dann eine erste Aufgabe an. Weitere Informationen finden Sie unter [F001 – Projektverwaltung](./F001-projektverwaltung.md) und [F002 – Aufgabenverwaltung](./F002-aufgabenverwaltung.md).

**Wie aktualisiere ich das Dashboard?**
Das Dashboard aktualisiert sich beim Öffnen automatisch. Sie können die Seite im Browser neu laden, um den aktuellsten Stand zu sehen. Eine manuelle Aktualisierungsschaltfläche ist nicht notwendig.

**Was bedeuten die Farben genau?**
🔵 Blau zeigt, dass die KI gerade aktiv arbeitet. 🟢 Grün bedeutet, die Aufgabe ist erfolgreich abgeschlossen. 🔴 Rot zeigt einen Fehler an. ⚪ Grau steht für offene oder noch nicht gestartete Aufgaben.

**Kann ich vom Dashboard direkt eine neue Aufgabe anlegen?**
Das Dashboard dient der Übersicht. Um eine neue Aufgabe anzulegen, navigieren Sie zunächst in das gewünschte Projekt und klicken dort auf **Neue Aufgabe**.

**Werden abgebrochene Aufgaben auch angezeigt?**
Ja, auch abgebrochene Aufgaben erscheinen im Dashboard, solange sie nicht gelöscht wurden. So behalten Sie den vollständigen Verlauf im Blick.
